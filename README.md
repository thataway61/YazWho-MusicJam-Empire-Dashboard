## ğŸš€ Getting Started

### **Quick Start**
1. **Access the Dashboard**: Visit the deployed application URL
2. **Connect GitHub**: Authorize your GitHub account
3. **Analyze Repository**: Enter any repository name
4. **Deploy Autonomously**: Click deploy and watch the magic happen

### **Local Development**
```bash
# Clone the repository
git clone https://github.com/thataway61/YazWho-MusicJam-Empire-Dashboard.git
cd YazWho-MusicJam-Empire-Dashboard

# Setup backend
cd backend
pip install -r requirements.txt
cp .env.example .env  # Add your API keys
python server.py

# Setup frontend
cd ../frontend
yarn install
yarn start
```

### **Environment Variables**
```bash
# Backend (.env)
MONGO_URL=mongodb://localhost:27017/empire_db
GITHUB_PAT=your_github_personal_access_token
GEMINI_API_KEY=your_gemini_api_key
GOOGLE_OAUTH_CLIENT_ID=your_oauth_client_id
GOOGLE_OAUTH_CLIENT_SECRET=your_oauth_client_secret

# Frontend (.env)
REACT_APP_BACKEND_URL=http://localhost:8001
```
